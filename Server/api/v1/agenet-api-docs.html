AgeNet API

API documentation for the AgeNet web client.

URL Prefix ¶

All endpoint URIs are implicitly prefixed with /agenet-api/api/v1 for berivity.

For example, /chemicals should be expanded to /agenet-api/api/v1/chemicals.

Validate Gene  ¶

Validate Gene
GET/genes/{database}/{gene}/validate

Validation endpoint; returns all available aliases for gene identifier supplied in either Symbol, Entrez or Ensembl formats. Matches partially typed identifiers.

A PPI object represents protein-protein interactions as a graph.

Protein-Tissue  ¶

Protein-Tissue
GET/ppi/genes

This endpoint returns a JSON object representing the nodes and edges of the interactions graph.

Example URI

GET /ppi/genes
URI Parameters
HideShow
database
string (required) Example: GTEx

GTEx

tissue
string (required) Example: all

Passing ‘all’ will return all genes with expressions above threshold, otherwise only genes that are expressed above threshold in tissue will be returned.

threshold
number (required) Example: 1.0

Only return genes expressing above this threshold. Defaults to 1.0 if none supplied.

array[string] (required) Example: ['ENSG00000162231', ...]

An array of genes in Ensembl format.
